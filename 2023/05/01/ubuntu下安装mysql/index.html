安装步骤

1、查看有没有安装MySQL：

若什么都没显示则说明没有安装

2、安装MySQL

apt install mysql-server

等待安装完毕

3、安装完成之后可以使用如下命令来检查是否安装成功：

netstat -tap | grep mysql

如果看到有 mysql 的socket处于 LISTEN 状态则表示安装成功。
